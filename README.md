# P2P-ET-in-Chile

This repository contains information about Blockchain and Simulation for Solar P2P Energy Trading in Chile



## Blockchain
The blockchain folder contains the files necessary to montain blockchain.

### Dependencies
- NodeJS
- Ganache
- Truffle
- Solidity


## Simulation
### System Requirements
- Processor: i5 equivalent or above
- Disk: Min 3 GB of free space
- RAM: Min - 4GB RAM. Recommended - 8 GB
- Graphics: Monitor resolution, Min 1024*768 Min. Optimal Scale and Layout setting - 100%
Operating system:
- Win 8.1, Win 10, Win 11, Win server 2016, and 2018. 64-bit only. Language: English.
  
Administrative rights required for installation
